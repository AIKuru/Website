# aydinkuru.com

Personal website of Aydın I. Kuru — researcher, electronic music practitioner and
PhD student at the University of Sussex.

Static site served by GitHub Pages on the custom domain **aydinkuru.com**
(DNS via Cloudflare).

## Pages
- `index.html` — landing / bio
- `bio.html` — biography, education, publications, performances
- `research.html` — research projects
- `compositions.html` — selected compositions
- `electronic.html` — electronic music / DJ
- `survey.html` — participant invitation for the tension study
- `404.html` — not-found page

## Assets
- `style.css` — shared styles
- `script.js` — ambient spectrum strip
- `favicon.svg`, `og-image.png` — icon and social-share card
- `files/` — downloadable PDFs (CV, consent form, participant information sheet, abstract)
- `CNAME` — custom domain configuration
